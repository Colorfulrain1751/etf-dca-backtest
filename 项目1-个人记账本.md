## 一、需求描述

市场部的小王每个月都不知道钱花哪了。我们需要开发一个**命令行记账工具**，第一版只要基础功能：

```
用户输入收入和支出 → 程序自动算出结余 → 给出简单分析
```

---

## 二、分天任务

### Day 1：让程序开口说话

**目标**：写出能和你交互的程序

**要学的东西**：

| 知识点 | 用在哪 |
|--------|--------|
| `print()` | 程序输出信息给用户看 |
| `input()` | 接收用户输入 |
| 变量 | 存储用户输入的数据 |
| `=` 赋值 | 把值放进变量里 |

**任务**：
1. 程序启动，打印一行字："=== 个人记账本 v1.0 ==="
2. 询问用户："本月收入是多少？"，用户输入一个数字
3. 询问用户："本月支出是多少？"，用户输入一个数字
4. 计算结余 = 收入 - 支出
5. 打印结果："本月结余：xxx 元"

**验收标准**：程序能跑通，输入 10000 和 6500，输出结余 3500。

---

### Day 2：让程序有点脑子

**目标**：给程序加上判断能力

**要学的东西**：

| 知识点 | 用在哪 |
|--------|--------|
| `if / elif / else` | 根据结余给出不同评价 |
| `int()` / `float()` | 把输入的字符串转成数字 |
| 比较运算符（`>` `<` `==`） | 判断结余情况 |

**任务**：
1. 在 Day 1 基础上，根据结余输出不同评价：
   - 结余 > 0：显示"👍 本月有结余，不错！储蓄率 xx%"
   - 结余 = 0：显示"⚠️ 月光族，一分不剩"
   - 结余 < 0：显示"🔴 本月超支 xx 元，注意控制！"
2. 计算储蓄率 = 结余 / 收入 * 100（保留 1 位小数）

**验收标准**：
- 输入收入 10000，支出 6500 → 显示结余 3500，储蓄率 35.0%
- 输入收入 5000，支出 5000 → 显示月光族提醒
- 输入收入 3000，支出 5000 → 显示超支 2000
